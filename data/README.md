# Data Sources

## signers.json
Discharge petition signers, fetched from the [Office of the Clerk](https://clerk.house.gov/DischargePetition/2026012215) via `scripts/parse-petition.mjs`.

## targeted.csv
Targeted representatives for outreach who have not yet signed. Phone numbers sourced from each member's official [House.gov](https://www.house.gov/representatives) page.
